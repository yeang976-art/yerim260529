# Backend Practice Archive

Java/Spring 백엔드 부트캠프 과정에서 진행한 실습, 학습 기록, GitHub 협업 흐름을 정리하는 저장소입니다.

이 저장소는 매일의 과제를 단순히 제출하고 끝내는 것이 아니라, 실습 과정에서 배운 개념과 문제 해결 과정을 기록해 백엔드 개발 포트폴리오로 발전시키기 위해 만들었습니다.

## What This Repository Contains

- 백엔드 학습 과정에서 작성한 실습 기록
- API, HTTP, Git, Java/Spring 관련 학습 정리
- Git branch, Pull Request, self review 기반의 협업 흐름 연습
- TIL, Notion, 기술 블로그와 연결되는 학습 아카이브

## Practice Areas

- API 설계와 요청/응답 흐름 분석
- Postman을 활용한 API 테스트와 Mock Server 실습
- Chrome DevTools를 활용한 네트워크 요청 분석
- Git/GitHub 기반 버전 관리와 협업 흐름
- Java/Spring 백엔드 개발 기초
- 데이터베이스와 서버 애플리케이션 구조

## Branch Structure

| Branch | Role |
| --- | --- |
| `main` | 최종 결과물이 병합되는 기본 브랜치 |
| `profile-foundation` | 기본 자기소개와 현재 학습 상태 |
| `backend-motivation` | 백엔드 개발자를 목표로 하는 이유 |
| `backend-practice-archive` | 부트캠프 실습과 과제 기록 |
| `git-collaboration-flow` | Git/GitHub 협업 흐름 정리 |
| `learning-system` | TIL, Notion, 기술 블로그 기반 학습 시스템 |

## Learning Direction

백엔드 개발은 사용자의 요청이 서버로 전달되고, 데이터가 처리되어 다시 응답으로 돌아오는 흐름을 설계하는 일이라고 생각합니다.

현재는 Java와 Spring을 중심으로 HTTP, API, 데이터베이스, Git 협업 흐름을 학습하고 있으며, 각 실습을 기록으로 남겨 실제 개발 역량을 설명할 수 있는 포트폴리오로 발전시키고 있습니다.

## Portfolio Goal

부트캠프가 끝나는 2026년 11월 18일까지 이 저장소를 실습 기록, 문제 해결 과정, 협업 경험이 연결된 백엔드 취업 포트폴리오로 발전시키는 것이 목표입니다.
